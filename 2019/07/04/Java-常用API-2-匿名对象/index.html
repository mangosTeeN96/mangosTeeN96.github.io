(Java)常用API--2.匿名对象

2019-07-04 java 常用API 评论

匿名对象：没有变量名的对象

格式:

new 类名(参数列表);

应用场景:

  1. 创建匿名对象直接调用方法，没有变量名
  2. 匿名对象只能使用唯一的一次，下次再用不得不再创建一个新对象。一旦调用两次方法，就是创建了两个对象，造成浪费
  3. 匿名对象可以作为方法的参数和返回值
